# Opathijian

Opathijian is a real-time multiplayer card game for 2 or more players, played with a standard 52-card deck without jokers.

Players compete to build the strongest hand by collecting cards of the same suit. The ultimate goal is to reach **41 points**, the highest possible score in the game.

This project recreates the traditional Opathijian experience as a web application, enabling players to create private rooms, join using room codes, and enjoy the game online from any modern browser on desktop or mobile devices.
